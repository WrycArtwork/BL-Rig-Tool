## 繁體中文
此插件主要為自定義Rigging設計，功能包括 **批量自訂骨骼顯示形狀**、**生成約束(BETA)**、**動作重定向** 以及 **批量重命名頂點組(Vertex Group)與形狀鍵(Shape Key)**。<br>
建議版本:Blender 4.2+ <br>
